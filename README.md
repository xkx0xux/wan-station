# わんステーション 公式サイト

滋賀発・保護犬支援プラットフォーム「わんステーション」（NPO法人わんステーション・ジャパン）の公式サイトです。
縦長トップ1枚＋将来ページの器（活動報告・FC加盟）で構成した、静的サイトです。

## 開き方（ローカルで見る）

`index.html` をダブルクリックしてブラウザで開くだけで表示できます。
（より正確に見たい場合はローカルサーバー経由で：ターミナルで以下を実行 → ブラウザで http://localhost:8000 ）

```
cd ~/わんステーション
python3 -m http.server 8000
```

## ファイル構成

```
わんステーション/
├── index.html        … トップ（縦長1枚・全セクション）
├── reports.html      … 活動報告（準備中の器）
├── franchise.html    … FC加盟（準備中の器）
├── css/style.css     … 全体の見た目（色・フォントはここの :root で一括管理）
├── js/main.js        … スマホメニュー・スクロール演出（無くても全内容は読めます）
├── assets/
│   ├── logo.svg      … ロゴ（仮・紺×ゴールドの肉球マーク）
│   └── img/          … 写真（すべて差し替え前提のフリー素材プレースホルダ）
└── docs/             … 設計書・実装プラン
```

## あとで差し替えるところ（本文中は 〔 〕 で表示）

- **Readyfor の公開URL** … 公開後に `index.html` の「Readyforで応援する」リンクへ
- **メールアドレス・SNS / YouTube のURL** … お問い合わせ欄とフッター
- **NPOの正式情報** … 所在地・代表者など（フッター）
- **実績数字** … ヒーローの「繋いだ命 〔128〕頭」
- **写真** … `assets/img/` の各画像を本物の写真へ
